Blood Group Identification

Introduction

This web app is used to find the blood group by uploading the blood cell image, and it will help to get the output as which blood group it is.

Blood Group Types

  • O+
  • O-
  • A+
  • A-
  • B+
  • B-

How to Use the Web App

1. Login with your registered credentials
2. Upload two types of blood cell images:

  • ABO blood cell image
  • Rh blood cell image
